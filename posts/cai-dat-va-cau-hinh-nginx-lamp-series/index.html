Cài đặt và cấu hình nginx

Categories: lamp

Tags: linux

  • Đăng nhập root
sudo su
  • Cài đặt NGINX sử dụng apt-get
sudo apt-get install nginx
  • Kiểm tra firewall
sudo ufw app list
  • Allow Nginx trên firewall
sudo ufw allow 'Nginx Full'
  • Kiểm tra trạng thái
systemctl status nginx
  • Add nginx to start up service
systemctl enable nginx
