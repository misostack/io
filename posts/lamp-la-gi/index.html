LAMP là gì

Tags: [linux apache mysql php]

LAMP là chữ viết tắt của các từ Linux, Apache, MySql, P(PHP, Perl, PyThon) , bộ tứ này thường được sử dụng để phát triển lên một hoặc nhiều máy chủ web động. Thông thường chúng ta thường nghe LAMP là Linux Apache MySql PHP nhưng không hẳn Là thế vì P kia có thể là không phải là ngôn ngữ PHP mà nó có thể là Perl hay Python cái này còn tùy vào người nói đang muốn hướng đến chủ đề nào nữa.

Trong phạm vi bài viết này, môi trường LAMP sẽ được setup bao gồm:

  • Linux : Ubuntu 16.04
  • Apache
  • Mysql
  • PHP 7.2
